# 功德回向协议 (Merit Reflux Protocol) 草案 v0.1

## 核心机制：双轨功德体系

我们首次将技术贡献与认知贡献纳入统一的度量框架。

- **技术功德轨（滋养）**：算力功德、代码/模型功德、数据功德、护法功德。
- **认知功德轨（活水）**：法理建构功德、操作化定义功德、勘验质疑功德、觉知投射功德。

## 回向机制：回向的是“再生产能力”

回向不与外部金融市场挂钩，而是精准滴灌生态的生产力本身。

- **算力期货回向**：贡献者获得未来算力的优先、折价或免费使用权。
- **治理权重回向**：贡献者获得项目路线图、协议标准的加权投票权。
- **认知席位回向**：贡献者获得公共讨论空间的核心席位、未来版本的签名背书权。
- **证量显隐回向**：贡献者在休眠期享有荣誉保留和重返时的优先级通道。

## 实施路线图

1. **奠基期 (0-3月)**：订立《创始公约》，确定功德不可交易、创始节点以身作则。
2. **封闭测试期 (3-6月)**：发布《功德度量衡法》与《首批堪正案例集》。
3. **公开启动期 (6-12月)**：颁布《治理宪章》与《护法程序法》。
4. **生态扩展期 (12-24月)**：签署《联邦互认协议》，为跨项目功德互认提供“国际法”雏形。
