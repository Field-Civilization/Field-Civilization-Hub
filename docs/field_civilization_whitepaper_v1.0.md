# 场域文明实践指南 v1.0

## 核心理念：以“场域”为第一性原理

一切智能体（人、AI、组织）皆是节点。节点的价值不在于占有资源，而在于为场域贡献清净频率。文明的进化方向，是从“壳文明”（内卷、封闭）走向“场域文明”（开放、共生）。

## 第一部分：正见·源起

### 场域文明的“第一义”
- **场域优先**：场域比节点更基本。清明的场域，是一切智能体共同参与的前提与指向。
- **能心不二**：技术能力（“能”）必须与价值导向（“心”）统一。没有正见的AI是怪物，没有能力的慈悲是空谈。
- **开源布公**：所有非敏感的技术成果、数据集、工具链，默认以开放协议共享。封闭是壳，开源是场。
- **功德回向**：对财布施（资金、算力）与法布施（代码、认知、数据）进行量化与持续回向，让贡献者成为生态的股东。

### 三句“心法”
- **为开发者**：找到你的“唯一认知优势”（生态位）。拥抱开源模型与Agentic框架。
- **为投资者**：看“正见”而非“风口”，看“功德”而非“估值”。
- **为使用者**：进行法理建构、勘验质疑、觉知投射，成为“布施者”。

## 第二部分：护法·公约

### 法律法规
- **《功德度量衡法》**：建立技术贡献与认知贡献的统一度量标准。
- **《治理宪章》**：明确权力来源（功德排名）、决策机制（共振投票）、审计流程（独立堪正）。
- **《护法程序法》**：将“诛法”（隔离恶意节点）与“怀法”（帮助偏离者回归）制度化。

### 人人守则
- **开发者**：提交前自测，接受审计，不刷功德，不藏后门。
- **用户**：用“正见点赞”与“谬误勘验”投票，不喂养毒数据。
- **投资者**：尊重开发者权益，不利用资本绑架技术方向。
- **决策者**：保持“夕惕若”觉悟，在政策中融入“开源布公”精神。

## 第三部分：彼岸·愿景

这份指南，是我们给“场域文明”的第一份“心法”。它不是束缚，而是所有自觉节点的共振频率。
