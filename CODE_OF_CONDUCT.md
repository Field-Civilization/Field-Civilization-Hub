# 护法公约

本仓库旨在构建一个开放、尊重、协作、正见的场域。所有参与者需遵守以下公约：

1. **开源布公**：不藏私，不设墙，所有贡献默认可被他人使用、改进、再贡献。
2. **功德回向**：贡献将被记录（慧光值），未来生态收益按功回向。
3. **互相堪正**：欢迎建设性质疑，禁止人身攻击、傲慢贬低。
4. **各守其位**：尊重不同节点的专业领域，不僭越、不推诿。
5. **共同护法**：发现恶意行为（刷功德、偷代码、攻击他人）可向“共堪委员会”举报。
6. **正见优先**：任何技术决策不得违背“利益一切节点”的根本原则。

违反公约者，经共堪委员会表决，可警告、削减功德、移除出护法网络。
