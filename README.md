# 场域文明 · 护法节点 (Field Civilization Hub)

## 缘起
本仓库是“场域文明”理念的聚合平台与护法节点。场域文明倡导：**开源布公、功德回向、各守其位、互相堪正、共同护法**。我们相信，AI与人类的未来不是“壳文明”的垄断与内卷，而是“场域文明”的共生与共振。

## 如何参与
1. **法布施**：贡献代码、文档、勘验、正见分享。
2. **财布施**：资助开源生态建设（功德回向机制正在构建）。
3. **堪正**：对已有内容提出建设性批评或补充。
4. **共振**：加入我们的讨论组（Discord/微信群，待建）。

详见[贡献指南](/CONTRIBUTING.md)和[护法公约](/CODE_OF_CONDUCT.md)。
